class qmAux.config_helper.Integration_weight(alias: str, cosine, sine, alt_code: Tuple[str, str] = None)¶

Bases: object

specification of a set of measurement integration weights. Result of integration will be:

sum over i of (W_cosine[i]cos[wt[i]] + W_sine[i]sin[wt[i]])analog[i]. Here: w is the angular frequency of the quantum element, and analog[i] is the analog data acquired by the controller. W_cosine, W_sine are the vectors associated with the ‘cosine’ and ‘sine’ keys, respectively. Note: the entries in the vector are specified in 4nsec intervals, and each entry is repeated four times during the demodulation.

Example: W_cosine = [2.0], W_sine = [0.0] will lead to the following demodulation operation: 2.0(cos[wt[0]]analog[0] + cos[wt[1]]analog[1] + cos[wt[2]]analog[2] + cos[wt[3]]analog[3])

>>> cfg.integration_weights['long_square_integW1'] = {# original syntax
...    'cosine': [1.0] * int(long_readout_ns / 4),
...    'sine': [0.0] * int(long_readout_ns / 4)
... }

property residuel_coef¶

This factor is to be multiplied to the qua variable VAR after calling qua.measure(...,VAR,...)

confer qx.meas_all_quadratures implementation for its use.

https://qm-docs.s3.amazonaws.com/v0.4/python/demod.html it is very important to make sure this section is well understood when using the demodulation operation, otherwise overflow may occur that will lead to corrupt results.

The simplest rule to remember to avoid overflow is as follows, valid when the demodulation is not very long: make sure the integration weights are such that when they multiply the raw ADC data, scaled to between -0.5 and 0.5, the result is between -2 and 2. :return:

qmAux.tools.format_results_meased(result: qm._results.JobResults, gs=0.0, Phis=0.0, amplification=1.0, fetch_idx='avg')¶

given job results, automatically return I,Q depending on which kind of streaming saves happened:

Possible modes of fetch_idx :

  • "avg" for default (post_measure_stream_processing() aka sa.buffer(*buffer).average().save('Ic'))

    >>> with qua.stream_processing():
>>>     qx.post_measure_stream_processing(prep_meas, (4,3,2))
------------ treated by ------------
>>> Ic = result.get('Ic').fetch_all()

  • "all" or "tag" for stream.save_all(tag) or direct tag saves qua.save(var, tag)

    >>> stream.save_all(tag)
>>> qua.save(var, tag)
------------ treated by ------------
>>> Ic = result.get('Ic').fetch_all()['value']

  • "fetcher" to get a generator object, not fully tested yet

    >>> jobresult.get('Ic').fetcher() # this is a generator object

  • some_slice_object for a certain range in the stream

    >>> stream.save_all(tag)
>>> qua.save(var, tag)
------------ treated by ------------
>>> result.get('Ic').fetch(some_slice_object)['value']
